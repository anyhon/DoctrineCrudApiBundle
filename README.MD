Installation
------------

1) Download using composer

```bash
composer require wernerdweight/doctrine-crud-api-bundle
```

2) Enable the bundle

Enable the bundle in your kernel:

```php
    <?php
    // config/bundles.php
    return [
        // ...
        WernerDweight\DoctrineCrudApiBundle\DoctrineCrudApiBundle::class => ['all' => true],
    ];
```
